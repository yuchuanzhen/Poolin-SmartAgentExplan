# Poolin智能代理基本功能运行脚本

---

***Windows***

## 硬件及系统要求

```asm
机器配置： 最低配置要求 2C 2G及以上，推荐 4C 4G及以上
系统要求： Windows10
```

---

## ETH 代理运行

- 创建目录
  
 ```asm
 C:/work/script
```

- 将对应的ETH代理脚本拷贝到上述目录
- 运行ETH代理脚本

```asm
 双击 C:/work/script/run_eth_proxy.cmd
```

- 查看日志

```asm
docker logs -f proxy_eth --follow --tail 10
```
